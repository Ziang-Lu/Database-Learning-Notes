## Operations

### Aggregation (聚集)

**Summarize multiple rows into a single row**

**(=> Compute a single value from a set of values)**

* Count   *(Any type)*

  Count the number rows in that colum that have the same value

* Sum   *(Number)*

* Avg   *(Number)*

* Max   *(Number)*

* Min   *(Number)*
